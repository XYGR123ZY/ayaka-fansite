# 神里绫华粉丝站 · 开发准备清单

> 在开始写代码之前，你需要准备好以下内容。  
> 标注 ⭐ 的是**必须项**，标注 🔧 的是**可选项**（影响功能范围但不阻塞开发）。

---

## 一、AI 相关 ⭐

### 1.1 LLM API Key（对话 + 记忆压缩）

绫华的"灵魂"——对话能力和记忆管理都依赖大模型。

| 选项 | 推荐模型 | 月费用估算 | 获取方式 |
| ---- | -------- | ---------- | -------- |
| **OpenAI** | GPT-4o-mini（日常对话，便宜） | ~$5-15/月 | [platform.openai.com](https://platform.openai.com) → API Keys |
| **Claude** | Claude 3.5 Haiku（备选） | ~$5-15/月 | [console.anthropic.com](https://console.anthropic.com) → API Keys |
| **国内模型** | DeepSeek / 通义千问 / 智谱 GLM | ¥10-50/月 | 各平台开发者中心 |

> **推荐**：先注册 OpenAI，用 GPT-4o-mini。成本低（约 ¥0.01/次对话），效果足够好。  
> **我需要**：API Key + 你选的模型名称（如 `gpt-4o-mini`）

### 1.2 AI 图片生成 API Key 🔧

影响功能：AI 生图模块（Phase 3）。不准备也不影响 MVP。

| 选项 | 费用 | 获取方式 |
| ---- | ---- | -------- |
| **Replicate** | 按量付费，~$0.01/张 | [replicate.com](https://replicate.com) → API tokens |
| **Stable Diffusion WebUI** | 自部署，免费（需要 GPU） | 本地或云 GPU 部署 |

> **我需要**：如果你现阶段就要 AI 生图 → 提供 Replicate API Token；否则可以先跳过。

### 1.3 AI 语音合成 🔧

影响功能：AI 语音朗读（Phase 2-3）。

| 选项 | 费用 | 获取方式 |
| ---- | ---- | -------- |
| **Edge TTS** | 免费 | 无需 API Key，直接调用微软免费接口 |
| **VITS 自部署** | 免费（需 GPU） | 后续自部署 |

> **推荐**：先用 Edge TTS，零成本零准备。如果绫华的声音效果不满意再换 VITS。

## 二、数据库 & 部署 ⭐

### 2.1 数据库

绫华的记忆存在数据库里，不是浏览器的 LocalStorage——所以需要真正的数据库。

| 阶段 | 方案 | 费用 | 获取方式 |
| ---- | ---- | ---- | -------- |
| **开发阶段** | SQLite（本地文件，零配置） | 免费 | 无需准备 |
| **生产环境** | Supabase（托管 PostgreSQL） | 免费额度 500MB | [supabase.com](https://supabase.com) 注册 → 创建项目 → 获取连接字符串 |
| 备选 | PlanetScale（MySQL 兼容） | 免费额度 5GB | [planetscale.com](https://planetscale.com) |

> **开发阶段不需要准备任何东西**，SQLite 本地自动运行。  
> 上线前再注册 Supabase 即可。

### 2.2 前端部署

| 平台 | 费用 | 获取方式 |
| ---- | ---- | -------- |
| **Vercel** | 免费额度充足 | [vercel.com](https://vercel.com) 用 GitHub 账号注册 |
| 备选 | Cloudflare Pages | 同上 |

> **我需要**：你的 GitHub 账号名称（用来关联 Vercel 自动部署）

### 2.3 后端部署

| 平台 | 费用 | 获取方式 |
| ---- | ---- | -------- |
| **Railway** | 免费额度 $5/月 | [railway.app](https://railway.app) 注册 |
| 备选 | Fly.io / 阿里云 ECS | — |

> **我需要**：Railway 账号（或你指定的其他部署平台）

## 四、素材资源 ⭐

> 这些不是"API Key"类的，但开发时必不可少。你准备得越完整，页面越像样。

### 4.1 绫华立绘 & 图片 🔧

| 类型 | 用途 | 建议来源 |
| ---- | ---- | -------- |
| 半身立绘（透明背景） | 首页氛围层 + 加载页 | 官方立绘 PNG，从 [Hoyolab Wiki](https://wiki.hoyolab.com/) 或 [Genshin Impact Wiki](https://genshin-impact.fandom.com/) 下载 |
| 官方立绘（多张） | 画廊"官方立绘"分类 | 同上 |
| 游戏截图 | 画廊"游戏截图"分类 | 你自己截，或从社区收集 |
| 绫华头像 | 聊天界面头像 | 裁剪立绘面部，128×128 |
| 加载图（多张） | B站式启动页轮换 | 选你喜欢的绫华图，建议 3-5 张，1080p+ |

### 4.2 音频素材 🔧

| 类型 | 用途 | 说明 |
| ---- | ---- | ---- |
| 稻妻 BGM | 首页氛围背景音乐 | 需要确认版权使用范围（个人项目一般问题不大） |
| 绫华语音片段 | 台词试听模块 | 游戏内语音提取（个人使用） |

### 4.3 字体 🔧

| 字体 | 用途 | 获取 |
| ---- | ---- | ---- |
| Noto Serif JP | 标题和风衬线 | [Google Fonts](https://fonts.google.com/) 免费 |
| 思源黑体 / 系统默认 | 正文 | 无需准备 |
